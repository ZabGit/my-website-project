## /Диаграмма прецедентов/

![Диаграмма прецедентов](Use%20Case.drawio.png)

- **Гость** → просматривает страницы, читает статьи  
- **Пользователь** → дополнительно оставляет комментарии, настраивает рекомендации и участвует в обсуждениях  
- **Администратор** → добавляет/удаляет статьи и управляет пользователями вдобавок к остальным правам

---

## /Диаграмма развёртывания/

![Диаграмма развёртывания](Deployment%20Diagram.drawio.png)

- **Браузер** (компьютер пользователя) ↔ **Сервер** (Apache, PHP, MySQL) через **HTTP**  
- **Сервер** ↔ **GitHub** (репозиторий кода) через **Git**
